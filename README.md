# Discord Bot

> self-hosted modular bot

This project's goal is to make a full featured, modular, self-hosted bot in javascript.

## Getting up and running
1. Clone this repo
2. `npm install`
3. Create a `.env` file with:
```
TOKEN=<Bot Token>
OWNER_ID=<Bot owner id>
```
4. `npm start`
